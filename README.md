Edited at 2023-01-26

喃字输入方案。

本输入方案使用telex模式。

输入规则：声母+韵母+韵尾，声调可以加在韵母中、韵母后或韵尾后。暂不支持直接改调。

后续版本会考虑支持改调和d变đ的功能，以及ươ判定等。

字母：đ=dd aa=â ee=ê oo=ô aw=ă ow=ơ uw=ư ooo=oo

声调：玄f(à) 锐s(á) 问r(ả) 跌x(ã) 重j(ạ)

支持两个简码：f=ph w=ư
